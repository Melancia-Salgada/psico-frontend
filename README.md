## 🚀 Instalando a EasyPsi

Para instalar a EasyPsi, siga estas etapas:

1. Após clonar os arquivos em sua máquina, abra o terminal dentro da pasta.
2. Utilize os seguinte comando para baixar as dependências.

```bash
npm install
```
